## Gradle

Jitpack for the win!

Step 1: Add jitpack to your root build.gradle at the end of repositories:

    allprojects {
		    repositories {
			  ...
			  maven { url 'https://jitpack.io' }
		    }
	    }

Step 2: Add the HtmlView2 dependency

	dependencies {
		compile 'com.github.stefanhaustein:htmlview2:v2.0.2'
	}
